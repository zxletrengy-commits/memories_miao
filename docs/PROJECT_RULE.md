\# Beyond the Stars



\## Project Position



Beyond the Stars is a personal interactive memory museum.



It is NOT a game remake.



It is NOT a fan game for public distribution.



It is a private local project created only to preserve memories after the official game is no longer available.



The experience should feel like entering a quiet café where memories still exist.



\---



\## Core Experience



The user should feel:



\- peaceful

\- nostalgic

\- warm

\- romantic

\- immersive



Everything should be slow.



Everything should feel alive.



No noisy UI.



No exaggerated effects.



\---



\## Design Philosophy



The website should never look like a normal webpage.



It should feel like the Home scene of a modern romance game.



Every interaction should be natural.



Every animation should be subtle.



Every transition should be elegant.



\---



\## Development Principles



Prefer reusable components.



Avoid hard-coded values.



Separate scene, character, interaction and data.



Keep every module independent.



Future content should be added by configuration instead of modifying logic.



\---



\## Supported Characters



Only two characters currently exist.



\- Qi Yu

\- Li Shen



Future expansion is allowed.



\---



\## Project Goal



Create a permanent digital place where memories can continue to exist.
